About me

My name is Alexander Tamarez and I’m a software developer based in New York. My whole life I thought I wanted to be a civil architect because it involves creating things and problem solving which are two things I’ve enjoyed doing for as long as I can remember. But back in 2013 I took an introduction to web development class by mistake and after realizing that there were so many things I could create with code and all the interesting challenges that come with programming, I changed my major to computer science and never looked back.

After graduating I started to teach myself web development and now I’m looking for a role as Front End Developer where I can help in the company’s growth and improve my skills as a professional web developer.
